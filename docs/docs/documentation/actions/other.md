# Duration Type

Sometimes, Action input is a String of type Duration.  
Expected format is `<positive number> <unit>`.  
`unit` values are :

* "nanos", "ns"
* "micros", "Âµs"
* "millis", "ms"
* "seconds", "s", "sec"
* "minutes", "m", "min"
* "hours", "h", "hour", "hours", "hour(s)"
* "days", "d", "day", "days", "day(s)"

**Examples** : "5 min", "300 sec", "1 day"
